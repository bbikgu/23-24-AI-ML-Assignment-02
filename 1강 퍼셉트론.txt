1강 퍼셉트론

인공신경망 : 생물학적인 신경망을 모방하여 만든 것.
신경망에 대한 이해 필요

Dendrite : 이웃 뉴런에서 전기 신호를 받는다
Synapse : 다른 뉴런과 Dendrite의 연결 부위에 있다. 전기 신호의세기를 재조정
Soma(cell body) : Dendrite로부터 받은 여러 전기신호들을 모두 합친다
Axon : Soma의 전위가 일정 이상이 되면 이웃 뉴런으로 전기 신호를 보냄(핵심적인 부분) 다음 시냅스에서 전기 신호를 재조정하여 받아짐.
옛날 봉화 시스템과 유사함.

퍼셉트론 : 신경망을 수학적으로 모델링하여 만든 것.
입력값에 가중치를 곱함. 그걸 합친 결과가 임계값을 넘지 않으면 0, 넘으면 1

퍼셉트론으로 AND게이트 구현
- 퍼셉트론에서 가중치, 임계값을 잘 결정해야함
-> 가중치, 임계값을 True, True -> True가 나오게 해야함. 나머지 경우는 False로 나오게 해야함

NAND게이트 구현
-> 이 역시 가중치, 임계값을 True, True -> False가 나오고 나머지 경우는 True로 나오게 해야함

OR게이트 구현
- False, False일때만 False가 나옴 -> 임계값을 낮게 설정해줌

XOR게이트 구현
- 서로 논리값이 다를 때만 True, 나머진 False
- 단층 퍼셉트론은 XOR게이트 구현 X
-> 다층 퍼셉트론 구현으로 해결(AND, NAND, OR게이트)
